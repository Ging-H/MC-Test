# MC-Test
st MC-SDK FOC5.4.3位置控制测试用工具

这是用于st MC-SDK 5.4.3的测试工具，通过串口发送控制指令，控制启动，停止，对齐，以及设置位置环的PID参数，目标值

#### 特别说明
* 5.4.3的位置控制需要先设置位置目标值,然后才能启动
* 多次发送0周期的位置目标值,电机将不会停止
* 发送周期不为0的位置目标值,实际上是以s型加减速运行,所以时间需要长一点.
